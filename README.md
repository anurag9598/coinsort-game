# 🪙 CoinSort Puzzle Game

CoinSort is a fun and challenging puzzle game where players must sort coins into the correct stacks. It’s simple to learn but tricky to master, offering a relaxing yet brain-teasing experience.

---

## 🎮 Features
- Intuitive drag-and-drop coin sorting mechanics
- Increasing difficulty with more stacks and coin types
- Clean and minimal UI for distraction-free gameplay
- Save and resume your progress
- Cross-platform support (desktop & mobile)

---

### Prerequisites

- npm or yarn package manager
